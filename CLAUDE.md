## 消逝时间功能说明

### 功能概述
CalCLI v1.1+ 新增消逝时间（expiration_time）功能，支持任务过期后自动跳过。

### 核心特性
1. **Task模型扩展**：新增expiration_time字段，默认值2121-02-01 21:21:00
2. **智能跳过**：如果当前日期在任务的消逝时间之后，完全跳过该任务
3. **命令支持**：
   - `create`命令：新增`--expiration`参数
   - `update`命令：支持更新`expiration_time`
4. **向后兼容**：旧数据自动添加默认消逝时间

### 一次性任务处理指南
对于一次性任务，推荐以下配置：
- **重复类型**：设置为`yearly`
- **重复间隔**：设置为`21`（每21年一次）
- **消逝时间**：设置为任务的终止时间
- **示例命令**：
  ```bash
  calcli create --name "一次性任务" --description "描述" \
    --begin "2026-03-27 09:00:00" --end "2026-03-27 10:00:00" \
    --repeat yearly --x 21 \
    --expiration "2026-03-27 10:00:00"
  ```
